Supported Variables

You can use the following variables in Command and Working Directory:

Variable Description
{{rootPath}} Project root directory path
{{workspaceFolder}} Current workspace folder path
$HOME User home directory
$USER Current username
$API_KEY API Key environment variable (must be set by user)
$(pwd) Current working directory (Shell command)

Examples:

echo $API_KEY
cd $HOME/projects/my-app
